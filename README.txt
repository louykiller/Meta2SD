Dependencias do projeto:
• org.webjars.webjars-locator-core:0.52 - Locate assets within WebJars
• org.webjars.sockjs-client:1.5.1 - SockJS Client
• org.webjars.stomp-websocket:2.3.4 - STOMP over WebSocket
• org.webjars.bootstrap:5.2.3 - Bootstrap CSS
• org.webjars.jquery:3.6.4 - jQuery
Ficheiro jsoup.jar que pode ser obtido em https://jsoup.org/

Como executar o projeto:
- Correr o ficheiro Googol.java
- Aceder a http://localhost:8080 no seu browser de preferência
